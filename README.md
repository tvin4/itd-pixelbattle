# Расширения для пиксель баттла ИТД nowkie 
Как как скачать и пользоваться - https://youtu.be/YPXOxab-r1g
Текстовая инструкция:
1. Чтобы пользоваться нужно скачать зип из релизов https://github.com/tvin4/itd-pixelbattle/releases/tag/v1
2. Распоковать в удобное место
3. Написать в адресную строку chrome://extensions/
4. Сверху справа включить режим разработчика
5. Сверху слева нажать "Загрузить распакованное расширение" и выбрать место куда вы распаковали папку
6. Расширение работает
